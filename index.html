About Lunarium Coin

Lunarium Coin is an innovative payment system, allowing anyone with internet access to receive or make payments to anyone and anywhere in the world. The network is built upon Masternodes as the governing hubs and Proof of Work as a validation method. There is no need for special equipment and central authority is nonexistent. Lunarium Coin is secure, private and instant.

Lunarium [XLN] is cryptocurrency used in Lunarium Coin system. It can be obtained by becoming a Masternode operator or mining on your CPU since it uses CPU friendly Quark algorithm. Another interesting way of obtaining XLN is by community involvement and helping project grow in form of promoting, translation, moderation, artwork, etc. which we highly value and reward.

Why Choose Us?

To follow the idea of truly decentralized system we have chosen following features for Lunarium:

Secure

Lunarium is based on the principles of cryptography, decentralization and consensus which ensures trustworthy transactions.

Private

PrivateSend is a trustless method of running a sequence of transactions such that an observer has difficulty tracing transaction history.

Cpu Mining

Quark algorithm so the process of distribution is fair among miners and everyone can mine following Bitcoin's original principle.

Masternodes

Masternodes enable PrivateSend, InstantSend and governance system features. Owners are rewarded for running Masternodes.

Instant Transactions

InstantSend technology enables transaction confirmation in just a few seconds through out the network.

Governance

Decentralized Governance by Blockchain. The system allows each Masternode to vote once for each proposal.

Is the source code available?
Yes! you can find us on GitHub here: Lunarium source code
How can I get Lunarium coins?
Lunarium can be obtained by process of mining, running a masternode or participating in community servers. Helping the Lunarium ecosystem by developing or adopting to existing apps is also rewarded.
How can I mine Lunarium coins?
In order to mine check miningpoolstats and pick a mining pool (consider keeping pool's under 51% overall hashrate for helping network stabilization). If you need more help check our #mining discord channel.
Where can I trade Lunarium coins?
Currently Lunarium can be traded on 4 exchanges (Vindax, Crex24 , Cratex and Altmarkets) and OTC in Discord (leave a ticket in OTC channel).
What is a masternode?
Masternodes are servers designed to provide advanced services and governance on the blockchain. Masternodes host full copies of the blockchain and provide a unique second layer of services to the network, facilitating advanced functions such as InstantSend and PrivateSend. Masternodes must be backed by collateral, and in return their operators receive regular payment for the services they provide to the network.
How can I run a masternode?
In order to start a masternode you need to have 100.000 ~ 250.000 XLN as collateral and a server where to run the software. There are two options available : Hosted masternode or Self-operated masternode. Both install methods are explained in the Masternode setup guide.
What other benefits are from owning a masternode?
Masternode operators are given the opportunity to vote each month on up to 10% of the block reward to fund community projects supporting the Lunarium ecosystem.
Is it a security ?
Absolutely not. Lunarium is an experimental cryptocurrency that it's not regulated or backed up by a company. You can send Lunarium to other people, receive some or exchange it at the markets. In the future, we will establish partnerships where it will be possible to exchange Lunarium for services.
